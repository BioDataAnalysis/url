    Filesystem Path Conversion¶

    Description¶

    The functions described below are convenience functions to convert between a URL and a filesystem path.

    Headers¶

    #include <skyr/filesystem/path.hpp>

    Example¶

    API¶

    Path¶

    auto skyr::v1::filesystem::from_path(const stdfs::path &path)¶

    Converts a path object to a URL with a file protocol. Handles some processing, including percent encoding

    Return

    a url object or an error on failure

    Parameters

    • path: A filesystem path

    auto skyr::v1::filesystem::to_path(const url &input)¶

    Converts a URL pathname to a filesystem path

    Return

    a path object or an error on failure

    Parameters

    • input: A url object
