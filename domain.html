    Domain Name Processing¶

    Description¶

    Punycode is an encoding which represents Unicode in the ASCII character set. It used for Internet domain names. It is specified in RFC 3482.

    The functions below apply conversions between Unicode encoded domain names and ASCII.

    Headers¶

    #include <skyr/domain/punycode.hpp>
#include <skyr/domain/domain.hpp>

    Example¶

    #include <skyr/domain/domain.hpp>
#include <iostream>

int main() {
  using namespace std::string_literals;

  auto domain = "उदाहरण.परीक्षा"s;
  auto encoded = skyr::domain::domain_to_ascii(domain);
  if (encoded) {
    std::cout << encoded.value() << std::endl;
  }
}

    API¶

    Punycode¶

    auto skyr::v1::punycode_encode(std::string_view input)

    Performs Punycode encoding based on a reference implementation defined in RFC 3492

    Return

    The encoded ASCII domain, or an error

    Parameters

    • input: A UTF-8 encoded domain to be encoded

    auto skyr::v1::punycode_decode(std::string_view input)¶

    Performs Punycode decoding based on a reference implementation defined in RFC 3492

    Return

    The decoded UTF-8 domain, or an error

    Parameters

    • input: An ASCII encoded domain to be decoded

    Domain to ASCII¶

    auto skyr::v1::domain_to_ascii(std::string_view domain, bool be_strict = false)

    Converts a UTF-8 encoded domain to ASCII using IDNA processing

    Return

    An ASCII domain, or an error

    Parameters

    • domain: A domain

    • be_strict: Tells the processor to be strict

    auto skyr::v1::domain_to_unicode(std::string_view ascii)¶

    Converts a Punycode encoded domain to UTF-8

    Return

    A valid UTF-8 encoded domain, or an error

    Parameters

    • ascii: A Punycode encoded domain
