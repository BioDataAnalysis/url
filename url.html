skyr::url¶

class url¶

Represents a URL. Parsing is performed according to the WhatWG specification

The API follows closely the WhatWG IDL specification.

For example: 

auto url = skyr::make_url("http://example.org/");
assert("http:" == url.value().scheme());
assert("example.org" == url.value().hostname());
assert("/" == url.value().pathname());
